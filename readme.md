## はじめに
このプロジェクトは、温湿度計を基板設計から作ろうという計画です。  
詳しくは下のブログを御覧ください。

## 回路設計
回路設計データは temp_monitor ディレクトリに入っています。  
KiCad 8.0で作成しています。なお、一部雑な部分があります。ご了承ください。  
なお、フットプリントは配布していません。ご了承ください。
